インストールと起動¶

インストール¶

インストールするにはMayaのスクリプトパスにgris3ディレクトリをコピーして下さい。

標準的なインストール先のディレクトリ¶

Windows

OSドライブ：\Users\<username>\Documents\maya\scripts

Linux

~/maya/scripts

Mac OSX

~/maya/scripts

起動¶

grisにはメインとなるファクトリーウィンドウの他、便利な機能を集めたツールバーなどがあります。起動するには以下のコマンドをスクリプトエディタのPythonタブにて実行して下さい。

ファクトリーウィンドウ¶

import gris3
gris3.showFactory()

Grisツールバー¶

import gris3
gris3.showToolbar()

その他ガジェットの起動¶

Grisツールバーは、名前の通りツールバースタイルのウィジェトですが、特定のツールを集中的に使うには不便な場合もあります。そのような場合は、特定のツールだけを切り出したガジェットを起動すると良いでしょう。

Joint Editor¶

from gris3 import gadgets
gadgets.openJointEditor()

参考

Joint Editor

モデリング用サポートウィンドウ¶

from gris3 import gadgets
gadgets.openModelSetup()

参考

Model Setup

ポリゴンを半分消去するウィンドウ¶

from gris3 import gadgets
gadgets.openPolyHalfRemover()

ポリゴンをミラーリングするウィンドウ¶

from gris3 import gadgets
gadgets.openPolyMirror()

ポリゴンをを任意の軸でカットするウィンドウ¶

from gris3 import gadgets
gadgets.openPolyCutter()

リネーマーを起動する¶

このコマンドは選択オブジェクトの数によってウィンドウが変化します。単体選択版と復数選択版が存在します。

from gris3 import gadgets
gadgets.showRenamer()
